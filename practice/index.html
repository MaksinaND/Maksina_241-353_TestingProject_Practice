Безопасность LAN

Базовая настройка сети Game Dev студии в PNETLab

Сетевая архитектура

Сегмент VLAN Подсеть Доступные IP Назначение
Программисты210.0.0.128/25.129 – .254Рабочие станции
Администраторы610.0.0.0/28.1 – .14Доступ к управлению
Серверы310.0.2.0/24.1 – .254DNS, DHCP, GIT, Syslog и т.д.
Интернет / WAN—192.168.10.0/24.1 – .254Подключение к внешней сети

Конфигурация оборудования

Базовая настройка сети включает в себя конфигурацию коммутаторов (Switch7, Switch8, Switch9) и маршрутизатора. Все настройки выполнены с учетом требований безопасности.

Конфигурация маршрутизатора

interface e0/0
 ip address 192.168.10.254 255.255.255.0
 no shutdown

interface e0/1.2
 encapsulation dot1Q 2
 ip address 10.0.0.1 255.255.255.128

interface e0/1.3
 encapsulation dot1Q 3
 ip address 10.0.2.254 255.255.255.0

Безопасность и защита данных

  • Port-Security для конечных устройств (только разрешенные MAC-адреса).
  • NAT и межвлан-маршрутизация на роутере.
  • Логирование на удаленный сервер SYSLOG (10.0.2.10).
  • Аудит и контроль доступа через ACL.
